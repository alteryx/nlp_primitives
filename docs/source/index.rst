==============================
What are NLP Primitives?
==============================

NLP Primitives are primitive functions created for use with Featuretools that take advantage of Natural Language Processing in order to create features for textual data. Some of these primitives are functions with multiple outputs that try to quantify the meaning of the text by vectorizing it, and some involve more basic functionas such as counting the number of capital letters or title words.
